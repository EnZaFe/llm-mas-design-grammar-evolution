# Visual Information

This directory contains all visual materials generated in the context of the companion paper, including figures, diagrams, and charts that illustrate the methodology, experimental setup, and results.

## Contents

- **Architecture diagrams** — Schematic representations of the LLM-based multi-agent system architectures explored and evolved in the study.
- **Grammar visualisations** — Graphical depictions of the context-free grammar used to define the MAS design space.
- **Evolutionary dynamics** — Plots showing fitness progression, population diversity, and convergence behaviour across generations.
- **Performance plots** — Bar charts, box plots, and comparative figures summarising the evaluation results on the target benchmarks.

## Formats

Figures are provided in both high-resolution raster format (PNG/PDF) for publication and editable vector format (SVG) where applicable. Plotting scripts are located in the `code/` directory.
